To view requests and responses for the REST API, use the developer tools in your browser.
For example, complete these steps to open some common developer tools:
  • Internet Explorer - Click Tools > F12 developer tools, click Network, and then click Start Capturing.
  • Firefox - Click Tools > Web Developer > Network.
  • Chrome - Open Chrome developer tools by pressing Ctrl+Shift+I, and then click the Network tab.
  • Safari - Open Safari Web Inspector by pressing Command-Option-I, and then click Timelines.
For detailed information about the API requests and responses, see the Data Load REST API Reference.
